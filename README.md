# Risk Dashboard (Python)

Built during a Master in Finance to bridge the gap between textbook risk theory and a working implementation тАФ covering the same KPIs a junior risk analyst would be expected to compute on day one.

A lightweight **market risk dashboard** built in Python to demonstrate core risk concepts (beyond compliance):

- **Prices**: BTC / FX / indices  
- **Returns**: simple or log  
- **Rolling volatility**: configurable window, annualized  
- **Historical VaR (95%)**: empirical quantile (loss magnitude)  
- **Stress scenario**: **-10%** shock expressed as **P&L** on a CHF notional  
- **Lookback window**: **1Y / 3Y / 5Y / 10Y / 20Y / MAX** (to compare regimes)

```powershell
.\.venv\Scripts\Activate.ps1
```

**macOS / Linux:**

## Download data (CSV)

The loader downloads market data via yfinance (Yahoo Finance) and saves CSV files into `data/raw/`.

```bash
python src/data_loader.py
```

Typical tickers:

- **BTC**: `BTC-USD`
- **FX (Swiss-relevant)**: `USDCHF=X`, `EURCHF=X`
- **Index**: `^SSMI` (SMI), fallback `^GSPC` (S&P 500)

> Note: data availability depends on the provider and ticker format.

## Methodology (risk)

### Returns

**Simple returns:**

$$r_t = \frac{P_t}{P_{t-1}} - 1$$

**Log returns:**

$$r_t = \ln(P_t) - \ln(P_{t-1})$$

### Rolling volatility

- Rolling standard deviation on daily returns
- Annualized with $\sqrt{N}$ where N = 252 (FX / indices) and N = 365 (crypto)

### Historical VaR (95%)

- VaR is computed from the historical return distribution
- Implemented as `quantile(alpha, method="lower")` тАФ takes the actual observed return at the threshold rather than interpolating between two observations
- Reported as a positive loss magnitude

> Example: if the 5th percentile return is тИТ3%, then VaR 95% = +3%.

### Stress scenario (тИТ10%)

- A simple instantaneous shock: `shock = тИТ10%`
- Converted to CHF P&L using a user-defined notional

---

## Assumptions & limitations

- Data comes from a public provider (Yahoo Finance via yfinance), so it may contain gaps or revisions.
- VaR is **not** Expected Shortfall (ES) and does not capture tail severity beyond the quantile.
- Historical VaR assumes returns are i.i.d. and drawn from a stationary distribution тАФ both assumptions break during crises.
- Stress test is a single-factor shock (no correlation / multi-asset portfolio yet).
